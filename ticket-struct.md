# пользователь вводит
1. дата
2. станция
3. id отправления (время, тип поезда, конечная станция)
4. если дальний поезд, тип вагона
5. фио
6. паспорт

# программа вычисляет
1. расстояние
2. стоимость
3. время в пути
4. время отправления
5. время прибытия
6. список станций по пути

# билет содержит
1. пассажир
   1. фио
   2. паспорт
2. отправление
   1. id отправления
   2. время отправления
   3. ~станция отправления~ (МСК)
   4. конечная станция
   5. тип поезда
3. станция прибытия
4. список станций по пути
5. расстояние
6. стоимость
7. время в пути
8. ~время прибытия~ (не хранить, а посчитать)
9.  тип вагона (если есть)
